# codename-waves
A Godot entry for Itch.io's Game Off 2025

- Concepts: https://www.merriam-webster.com/dictionary/wave
    - Nouns:
        - Waves of a beach (or any fluid)
        - Waves of enemies (e.g. zombies)
        - Synthwave
        - Sound waves
        - Waves of trigonometric functions
    - Verbs:
        - Waving your hand

- Ideas:
    - Guitar Hero x Beat Saber, but 8-bit music
        - Notes of the song playing in the background come at you as waves
    - Game where you skip stones to score points based on (wave size?)
    - Soundswaves as a weapon?
        - Different types of waves have different properties
        - Enemies come in waves, and you have to create a type of wave that matches the enemy pattern
    - Game that helps you learn to make chiptune music by showing and playing audio for a sound wave and you have to adjust parameters to match the wave
        - Higher difficulties add more parameters
        - Score more points the faster you solve
